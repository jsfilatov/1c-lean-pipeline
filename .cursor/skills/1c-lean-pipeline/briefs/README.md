# Брифы для субагентов (`1c-lean-pipeline`)

Использование: оркестратор открывает соответствующий `brief-*.md`, подставляет плейсхолдеры из таблицы в `SKILL.md` (в т.ч. `{{HANDOFF_JSON}}`, опционально `{{HANDOFF_RESPONSE_JSON}}`), дополняет блоком «Каталог задачи / обязательно прочитай / пиши результат».

Перед вызовом субагента материализуй **HandoffRequest** в `{{TASK_DIR}}/handoff/request-*.json` по схеме `../schemas/handoff-request.schema.json` (см. раздел «JSON handoff» в `SKILL.md`). В тело вызова `Task` передавай **путь к файлу**, а не весь JSON.

**Не редактировать** файлы в `.cursor/agents/*` под этот процесс — контракт задаётся здесь и в `SKILL.md` (исключения по проекту — см. корневой `README.md` навыка).

**Токены:** в контекст вызова подмешивать **только один** `brief-*.md` на одного субагента за итерацию; не прикреплять ко всем вызовам весь каталог `briefs/`.
